## Lancer

Pour lancer via vscode le projet, dupliquez le fichier *launch.json.default* dans */.vscode/* et enlevez l'extension *.default* de ce fichier dupliqué.<br>
Dans l'onglet **Run and Debug**, choisir la bonne configuration, **Launch** ici pour lancer, le raccourci par défaut devrait être *Shift+F5*.<br>
Si vous voulez faire des modifs personnelles à cette configuration, modifiez le fichier *launch.json* seulement, sauf si vous pensez que cette configuration peut être utile à tout le monde.<br>
